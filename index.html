Введение

Зачем нужны
математические модели?

Экология изучает взаимодействие живых организмов с окружающей средой. Наблюдений недостаточно — нужна математика.

01
Наблюдение
Нельзя пересчитать всех волков в лесу или все бактерии в колонии. Модель дополняет неполные данные и выявляет закономерности, невидимые глазу.
02
Предсказание
Зная уравнение, можно предсказать численность популяции через год и через десять лет — и вовремя предотвратить вымирание вида.
03
Управление
Математика даёт инструменты: когда вводить охрану, когда разрешать охоту, сколько рыбы выловить без ущерба для экосистемы.
Модель 01

Линейная модель
популяции

// Основное уравнение
\(\frac{dx}{dt} = A - B\)
A — рождаемость, B — смертность особей в единицу времени
// Линейный случай: A = ax, B = bx
\(\frac{dx}{dt} = (a-b)\cdot x\)
a — коэффициент рождения, b — коэффициент смерти
// Решение
\(x(t) = x_0 \cdot e^{(a-b)(t-t_0)}\)
Классическое экспоненциальное решение методом разделения переменных

Показатель степени \((a-b)(t-t_0)\) определяет судьбу популяции:

Рост популяции
Если коэффициент рождения превышает смертность, популяция растёт неограниченно. Пример: бактерии в питательной среде.
\(a > b \Rightarrow x(t) \to \infty\)
Вымирание
Если смертность превышает рождаемость, популяция экспоненциально убывает к нулю. Вид оказывается под угрозой исчезновения.
\(a < b \Rightarrow x(t) \to 0\)
Вывод решения
0
Исходное уравнение
Из модели: \(A=ax,\; B=bx\), подставляем в \(\frac{dx}{dt}=A-B\):
\(\dfrac{dx}{dt} = (a-b)\cdot x\)
1
Разделение переменных
Делим обе части на \(x\), умножаем на \(dt\):
\(\dfrac{dx}{x} = (a-b)\,dt\)
2
Интегрирование
Интегрируем обе части:
\(\ln|x| = (a-b)\,t + C\)
3
Потенцирование
Применяем \(e^{(\cdots)}\) к обеим частям, обозначаем \(e^C = A_0\):
\(x(t) = A_0 \cdot e^{(a-b)t}\)
4
Начальное условие x(t₀) = x₀
Из \(x_0 = A_0 e^{(a-b)t_0}\) находим \(A_0 = x_0 e^{-(a-b)t_0}\)
// Решение
\(x(t) = x_0 \cdot e^{(a-b)(t-t_0)}\)
\(a>b\) → Рост: показатель \(>0\), \(e^{(\cdots)}\!\nearrow\), \(x(t)\to\infty\)

\(t=t_0\): \(e^0=1\), значит \(x(t_0)=x_0\) ✓
Модель 02

Логистическая модель:
ёмкость среды

В реальном мире ресурсы ограничены. Добавляем нелинейный «тормоз» −bx², и популяция больше не уходит в бесконечность.

// Нелинейное уравнение
\(\frac{dx}{dt} = ax - bx^2\)
−bx² — «тормоз»: чем больше популяция, тем сильнее конкуренция
// Решение (формула 33)
\(x(t) = \dfrac{x_0 \cdot \tfrac{a}{b}}{x_0 + \!\left(\tfrac{a}{b} - x_0\right)e^{-a(t-t_0)}}\)
При \(t\to\infty\): \(x(t)\to \tfrac{a}{b} = K\) (несущая ёмкость среды)
  • Если \(x_0 < K\): S-образный рост — сначала ускорение, потом торможение
  • Если \(x_0 > K\): плавный спуск к тому же равновесию \(K\)
  • \(K = \tfrac{a}{b}\) — универсальная константа: популяция всегда стремится к ней
Вывод формулы
1
Разделение переменных
Делим на \(x(a-bx)\), умножаем на \(dt\):
\(\dfrac{dx}{x(a-bx)} = dt\)
2
Разложение на простые дроби
\(\frac{1}{x(a-bx)} = \frac{A}{x} + \frac{B}{a-bx}\); решая: \(A=\frac{1}{a},\; B=\frac{b}{a}\)
\(= \dfrac{1}{a}\cdot\dfrac{1}{x} + \dfrac{1}{a}\cdot\dfrac{b}{a-bx}\)
3
Интегрирование
\(\dfrac{1}{a}\ln|x| - \dfrac{1}{a}\ln|a-bx| = t + C\)
4
Упрощение логарифмов → потенцирование
\(\ln\!\tfrac{x}{a-bx} = at + C_1\), затем \(e^{(\cdots)}\):
\(\dfrac{x}{a-bx} = K\,e^{at}\)
5
Начальное условие → выражаем x(t)
\(K = \tfrac{x_0}{a-bx_0}e^{-at_0}\), подставляем и выражаем \(x\):
// Формула (33) — логистическое решение
\(x(t) = \dfrac{x_0 \cdot \tfrac{a}{b}}{x_0 + \!\left(\tfrac{a}{b} - x_0\right)e^{-a(t-t_0)}}\)
При \(t\to\infty\): \(e^{-a(\cdots)}\to 0\), \(\,x(t)\to\tfrac{a}{b}=K\) ✓
Модель 03

Хищник — Жертва
Лотка–Вольтерра

Альфред Лотка и Вито Вольтерра независимо открыли одну и ту же систему уравнений, объяснившую колебания рыбных уловов в Адриатическом море.

1925
// Уравнение хищников
\(\frac{dx}{dt} = -ax + bxy\)
Хищники \(x\). Без жертв (\(y=0\)): убывают со скоростью \(-ax\). Встречи \(+bxy\) — пропитание и рост.
// Уравнение жертв
\(\frac{dy}{dt} = cx - dxy\)
Жертвы \(y\). Без хищников растут: \(+cx\). Встречи \(-dxy\) сокращают численность.
1
Жертв много
Корма достаточно — хищники питаются хорошо и начинают размножаться
2
Хищников больше
Давление на жертв усиливается — их численность начинает падать
3
Жертв мало
Хищники голодают — их численность тоже начинает сокращаться
4
Цикл повторяется
Жертвы восстанавливаются — и вся история начинается снова
Фазовая динамика

Колебания во времени

Хищники и жертвы колеблются с одним периодом, но со сдвигом фаз — максимум жертв всегда предшествует максимуму хищников

v(τ) — жертвы
u(τ) — хищники
Фазовый анализ

Первый интеграл
и фазовые кривые

// Безразмерная форма (u, v, τ)
\(u' = \alpha u(v-1)\)
\(v' = v(1-u)\)
u = (d/c)x, v = (b/a)y, τ = ct, α = a/c
// Первый интеграл системы
\(\alpha v + u - \ln(v^\alpha u) = H\)
\(H\) — константа начальных условий. Каждому \(H\) — своя замкнутая кривая. \(\tfrac{dH}{d\tau}=0\).
  • Малый H → маленький цикл вблизи равновесия (1,1)
  • Большой H → широкий цикл, сильные колебания популяций
  • Траектория не может перескочить с одной кривой на другую — H сохраняется
Вывод первого интеграла
1
Безразмерная форма
Замена \(u=\tfrac{d}{c}x,\; v=\tfrac{b}{a}y,\; \tau=ct,\; \alpha=\tfrac{a}{c}\) сводит 4 параметра к одному:
\(u' = \alpha u(v-1),\quad v' = v(1-u)\)
2
Исключаем время τ
Делим \(u\') на \(v\'\) — время \(\tau\) исчезает, остаётся уравнение траектории:
\(\dfrac{du}{dv} = \dfrac{\alpha u(v-1)}{v(1-u)}\)
3
Разделение переменных
Переносим: всё с \(u\) влево, всё с \(v\) вправо:
\(\dfrac{1-u}{u}\,du = \alpha\,\dfrac{v-1}{v}\,dv\)
4
Интегрирование обеих частей
\(\int\!\left(\tfrac{1}{u}-1\right)du = \ln u - u\),\quad \(\int\!\left(\alpha-\tfrac{\alpha}{v}\right)dv = \alpha v - \alpha\ln v\)
\(\ln u - u - \alpha v + \alpha\ln v = -H\)
5
Первый интеграл
Умножаем на \(-1\), используем \(\alpha\ln v = \ln(v^\alpha)\):
// Инвариант Лотки–Вольтерры
\(\alpha v + u - \ln(v^\alpha u) = H\)
\(H\) не меняется со временем: \(\tfrac{dH}{d\tau}=0\) ✓  Каждое \(H\) задаёт одну замкнутую орбиту. Начальные условия \((u_0, v_0)\) фиксируют \(H\) и конкретный цикл.
