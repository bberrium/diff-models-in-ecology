Как математика описывает жизнь — рост популяций, конкуренцию видов и вечные циклы природы. Три модели: от простого к сложному.

\(\dfrac{dx}{dt} = (a - b)\,x\)
Знак (a−b) определяет: рост или вымирание
// Модель 02 · Логистическая
\(\dfrac{dx}{dt} = ax - bx^2\)
Популяция стремится к несущей ёмкости \(K = a/b\)
// Модель 03 · Хищник (Лотка–Вольтерра)

y — жертвы; без хищников растут по \(+cy\)
Введение

Зачем нужны
математические модели?

Экология изучает взаимоотношения живых организмов с окружающей средой. Основным объектом является эволюция популяций. Наблюдений недостаточно — нужна математика.

01
Наблюдение
Нельзя пересчитать всех волков в лесу или все бактерии в колонии. Модель дополняет неполные данные и выявляет закономерности, невидимые прямому наблюдению.
02
Предсказание
Зная уравнение, можно предсказать численность популяции через год и через десять лет — и вовремя предотвратить вымирание вида или вспышку вредителей.
03
Управление
Математика даёт инструменты: когда вводить охрану, когда разрешать охоту, сколько рыбы выловить без ущерба для экосистемы.
Модель 01

Линейная модель
популяции

// Базовое уравнение (30)
\[\frac{dx}{dt} = A - B\]
\(x\) — число особей в популяции в момент \(t\)
\(A\) — число рождающихся в единицу времени
\(B\) — число умирающих в единицу времени
// Линейный случай (31): A = ax, B = bx
\[\frac{dx}{dt} = (a - b)\,x\]
\(a\) — коэффициент рождения, \(b\) — коэффициент смерти (на особь в единицу времени)
// Решение (32) при начальном условии x(t₀) = x₀
\[x(t) = x_0\,e^{(a-b)(t-t_0)}\]
Классическое экспоненциальное решение. Получается методом разделения переменных.
1
Разделяем переменные
Делим обе части на \(x\), переносим \(dt\) вправо:
\[\frac{dx}{x} = (a-b)\,dt\]
2
Интегрируем обе части
\[\int_{x_0}^{x} \frac{d\xi}{\xi} = \int_{t_0}^{t}(a-b)\,d\tau\]
3
Вычисляем интегралы
\[\ln x - \ln x_0 = (a-b)(t - t_0)\]
4
Потенцируем — итог
\[x(t) = x_0\,e^{(a-b)(t-t_0)}\]
Показатель (a−b)(t−t₀) определяет поведение:
· \(a > b\) → показатель > 0 → \(x(t) \to \infty\) при \(t \to \infty\)
· \(a < b\) → показатель < 0 → \(x(t) \to 0\) при \(t \to \infty\)
· \(a = b\) → \(x(t) = x_0 = \text{const}\) (равновесие)

Хотя модель является упрощённой, она в ряде случаев соответствует действительности — например, для фруктовых вредителей и некоторых видов бактерий.

Рост популяции
Если рождаемость превышает смертность — популяция растёт неограниченно. Пример: бактерии в питательной среде, плодовые мушки.
\(a > b \;\Rightarrow\; x(t) \to +\infty\)
Вымирание
Если смертность превышает рождаемость — популяция экспоненциально убывает к нулю. Вид оказывается под угрозой.
\(a < b \;\Rightarrow\; x(t) \to 0\)
Модель 02

Логистическая модель:
ёмкость среды

Практически все модели, описывающие реальные явления, нелинейны. Вместо линейного уравнения рассматривается нелинейное, где \(-bx^2\) — «тормоз» перенаселённости.

// Нелинейное уравнение
\[\frac{dx}{dt} = ax - bx^2 = x(a - bx),\quad a,b > 0\]
Слагаемое \(-bx^2\) — конкуренция: чем больше популяция, тем сильнее внутривидовая борьба за ресурс
1
Разделяем переменные
Делим обе части на \(x(a-bx)\), умножаем на \(dt\):
\[\frac{dx}{x(a-bx)} = dt\]
2
Разложение на простые дроби
Используем метод частичных дробей:
\[\frac{1}{x(a-bx)} = \frac{1}{a}\!\left(\frac{1}{x} + \frac{b}{a-bx}\right)\]
3
Интегрируем обе части
\[\frac{1}{a}\int\!\left(\frac{1}{x}+\frac{b}{a-bx}\right)dx = \int dt\]
4
Вычисляем интегралы
\[\frac{1}{a}\!\left(\ln x - \ln(a-bx)\right) = t + C_1\]
5
Потенцируем
\[\frac{x}{a-bx} = K\,e^{at},\quad K = \text{const}\]
6
Начальное условие \(x(t_0)=x_0\)
Подставляем \(t=t_0,\; x=x_0\):
\[K = \frac{x_0}{a-bx_0}\,e^{-at_0}\]
7
Выражаем \(x\) из уравнения
Подставляем \(K\) обратно, группируем экспоненту \(e^{a(t-t_0)}\):
\[\frac{x}{a-bx} = \frac{x_0}{a-bx_0}\,e^{a(t-t_0)}\]
8
Итог — формула (33)
Вводим несущую ёмкость \(K = a/b\):
\[x(t) = \frac{x_0\cdot(a/b)}{x_0 + \bigl(\tfrac{a}{b}-x_0\bigr)\,e^{-a(t-t_0)}}\]
// Точное решение (33)
\[x(t) = \frac{x_0\cdot K}{x_0 + (K-x_0)\,e^{-a(t-t_0)}},\quad K=\frac{a}{b}\]
При \(t\to\infty\): \(x(t)\to K = a/b\) — устойчивое равновесие. Несущая ёмкость среды \(K\).
  • Если \(x_0 < K\): S-образный логистический рост — ускорение, затем торможение
  • Если \(x_0 > K\): монотонный спуск к тому же равновесию \(K\)
  • \(K = a/b\) — популяция всегда стремится к этому значению независимо от начальных условий

aкоэффициент рождаемости
bкоэффициент конкуренции
K = a/bнесущая ёмкость среды
x₀начальная численность
Модель 03

Хищник — Жертва
Лотка–Вольтерра

Альфред Лотка (1925) и Вито Вольтерра (1926) независимо построили одну и ту же систему уравнений — для объяснения колебаний уловов рыбы в Адриатическом море.

1926
// Уравнение хищников (34) · x — большие рыбы
\[\frac{dx}{dt} = -ax + bxy\]
Без жертв (\(y=0\)): \(\frac{dx}{dt} = -ax < 0\) — хищники убывают.

Слагаемое \(+bxy\) — прирост хищников от «встреч» с жертвами (частота встреч \(\propto xy\)).
// Уравнение жертв (35) · y — малые рыбы
\[\frac{dy}{dt} = cy - dxy\]
Без хищников (\(x=0\)): \(\frac{dy}{dt} = cy > 0\) — жертвы растут.

Слагаемое \(-dxy\) — убыль жертв в зависимости от численности хищников.
ПараметрСмыслЗнак
aсмертность хищника без пищи+
bэффективность хищника (прирост от встречи)+
cрождаемость жертвы без хищника+
dинтенсивность выедания жертвы+
1
Жертв много
Корма достаточно — хищники питаются хорошо и начинают активно размножаться
2
Хищников больше
Давление на жертв усиливается — их численность начинает резко падать
3
Жертв мало
Хищники голодают — их численность тоже сокращается по закону \(-ax\)
4
Цикл повторяется
Жертвы восстанавливаются — и вся история начинается снова. Колебания не затухают.
Временна́я динамика · Рис. 8

Колебания популяций во времени

При \(v_0 > 1\) и \(u_0 < 1\): максимум жертв \(v(\tau)\) всегда предшествует максимуму хищников \(u(\tau)\). Колебания происходят в разных фазах.

v(τ) — жертвы (малые рыбы)
u(τ) — хищники (большие рыбы)
\(u = \tfrac{d}{c}\,x\;\cdot\; v = \tfrac{b}{a}\,y\;\cdot\; \tau = ct\;\cdot\; \alpha = a/c\)
Безразмерные переменные (36), введённые для удобства исследования
Фазовый анализ

Первый интеграл
и фазовые кривые

// Безразмерная форма системы (36)
\[u' = \alpha\,u\,(v-1),\qquad v' = v\,(1-u)\]
Штрих — дифференцирование по \(\tau\). \(\alpha > 0\).
Равновесие системы: \(u=1,\; v=1\).
1
Делим уравнения друг на друга
Делим первое уравнение системы (36) на второе:
\[\frac{du}{dv} = \frac{\alpha\,u\,(v-1)}{v\,(1-u)}\]
2
Разделяем переменные
\[\frac{1-u}{u}\,du = \alpha\,\frac{v-1}{v}\,dv\]
3
Интегрируем обе части
\[\int\!\frac{1-u}{u}\,du = \ln u - u + C_1\]
\[\alpha\!\int\!\frac{v-1}{v}\,dv = \alpha(v - \ln v) + C_2\]
4
Перегруппировываем — первый интеграл
Объединяем константы, определяемые начальными условиями:
\[\alpha v + u - \ln(v^\alpha \cdot u) = H = \text{const}\]
// Первый интеграл системы
\[\alpha v + u - \ln(v^\alpha \cdot u) = H\]
\(H\) — постоянная, определяемая начальными условиями (37) и параметром \(\alpha\).
Каждому значению \(H\) соответствует своя замкнутая кривая.
\(H\) не меняется со временем: \(dH/d\tau = 0\) — закон сохранения системы.
  • Малый \(H\) → маленький цикл вблизи равновесия (1, 1)
  • Большой \(H\) → широкий цикл, сильные колебания обеих популяций
  • Траектория не перескакивает с одной кривой на другую — \(H\) сохраняется
  • Начальные условия \(u_0, v_0\) однозначно задают конкретный цикл через \(H_3\)
Расширение модели

Конкуренция видов:
Принцип Гаузе

Если две популяции конкурируют за один и тот же источник питания (третья популяция), можно показать: один из видов неизбежно начнёт вымирать.

// n сосуществующих видов — система (общий вид)
\[\frac{dx_i}{dt} = f_i(x_1,\ldots,x_n),\quad i=1,2,\ldots,n\]
Каждый вид описывается своим уравнением в системе
// Принцип конкурентного исключения Гаузе (1934)
Один вид → один ресурс → вымирание конкурента
Если два вида конкурируют за один источник питания, один из них неизбежно вытесняет другого. Если один становится источником питания для третьей популяции — та же участь постигает и остальных.
  • Изучение сообществ, взаимодействующих более сложным образом, даёт более интересные практические результаты
  • При конкуренции за разные ресурсы виды могут устойчиво сосуществовать

Итоги

Что мы изучили

Линейная модель
Знак разности \((a-b)\) определяет судьбу: рост при \(a > b\), вымирание при \(a < b\). Решение: \(x(t)=x_0 e^{(a-b)(t-t_0)}\).
Логистическая модель
Слагаемое \(-bx^2\) ограничивает рост. Популяция стремится к ёмкости среды \(K = a/b\) по S-образной кривой. Взрыва не происходит.
Многовидовые системы
Каждый вид описывается своим уравнением: \(\dfrac{dx_i}{dt} = f_i(x_1,\ldots,x_n)\).
Лотка–Вольтерра
Хищник и жертва образуют замкнутые периодические циклы. Колебания не затухают. Исторически подтверждены данными по рыболовству в Адриатике.
Первый интеграл
\(\alpha v + u - \ln(v^\alpha \cdot u) = H\) — инвариант системы. Начальные условия задают конкретный замкнутый цикл через константу \(H\).
Принцип Гаузе
При конкуренции двух видов за один ресурс один из видов неизбежно вымирает. Математически строгий вывод из системы дифференциальных уравнений.
